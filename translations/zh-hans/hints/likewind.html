我不知道我应该如何处理这个问题。

你需要做2个简单的步骤：从末尾找到第1、第3和第2个元素，然后把它们作为一个元组返回。请注意，在Python中，元素的编号是从0开始的，而不是1。

你能给出一个说明性的例子吗？

例如，我们有一个元组('a', 'b', 'c', 'd', 'e', 'f', 'g') ，我们需要找到它的第一、第三和第二元素。当从一个元组的开始计算元素时，第一个元素的索引是0。如果从尾部开始计算，"第一个 "元素的索引是"-1"，第二个是"-2"，以此类推。代码将如下： 

elements = ('a', 'b', 'c', 'd', 'e', 'f', 'g')
x = elements[0]
y = elements[2]
z = elements[-2]

之后，x 将取值'a',y -'c',z -'f' 。

如何将得到的结果转换成一个元组？

不需要特别的操作。你可以立即使用return 。 

def easy_unpack(elements: tuple) -> tuple:

    x = elements[0]
    y = elements[2]
    z = elements[-2]
    
    return x, y, z

顺便说一下，如果不存在与其他变量混淆的危险，就没有必要将元组用括号括起来。该解决方案可以写得更简单： 

def easy_unpack(elements: tuple) -> tuple:

    return elements[0], elements[2], elements[-2]
