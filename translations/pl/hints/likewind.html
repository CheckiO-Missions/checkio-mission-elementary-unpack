Nie wiem jak powinienem podejść do tego problemu.

Musi Pan wykonać 2 proste kroki: znaleźć 1, 3 i 2 element od końca, a następnie zwrócić je jako krotkę. Proszę zwrócić uwagę, że w Pythonie numeracja elementów zaczyna się od 0, a nie od 1.

Czy mógłby Pan podać ilustrujący przykład?

Na przykład mamy krotkę ('a', 'b', 'c', 'd', 'e', 'f', 'g') i musimy znaleźć jej pierwszy, trzeci i drugi element od końca. Licząc elementy od początku krotki - pierwszy element będzie miał indeks 0. Jeśli liczymy od końca, "pierwszy" element będzie miał indeks "-1", drugi - "-2" i tak dalej. Kod wyglądałby następująco: 

elements = ('a', 'b', 'c', 'd', 'e', 'f', 'g')
x = elements[0]
y = elements[2]
z = elements[-2]

Po czym x przyjmie wartość 'a', y - 'c', z - 'f'.

Jak przekonwertować uzyskane wyniki na krotkę?

Nie są wymagane żadne specjalne działania. Mogą Państwo natychmiast skorzystać z return. 

def easy_unpack(elements: tuple) -> tuple:

    x = elements[0]
    y = elements[2]
    z = elements[-2]
    
    return x, y, z

Nawiasem mówiąc, nie jest konieczne ujmowanie krotki w nawiasy, jeśli nie ma niebezpieczeństwa pomylenia jej z innymi zmiennymi. Rozwiązanie można napisać jeszcze prościej: 

def easy_unpack(elements: tuple) -> tuple:

    return elements[0], elements[2], elements[-2]
