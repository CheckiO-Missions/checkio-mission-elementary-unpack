Я не знаю, як взятися за цю проблему.

Ти маєш зробити два простих кроки: знайди перший, третій і другий з кінця елементи, і поверни їх як кортеж. Будь уважний, у Python номери елементів починаються з 0, а не з 1.

Чи можеш ти показати на прикладі?

Наприклад, ми маємо кортеж ("a", "b", "c", "d", "e", "f", "g") і нам потрібно знайти 1-й, 3-й і 2-й з кінця елементи. Коли рахуємо елементи з початку кортежу - перший елемент матиме індекс 0. Якщо ж рахуємо з кінця, то "перший" елемент матиме індекс "-1", другий - "-2" і так далі. Код матиме наступний вигляд: 

elements = ("a", "b", "c", "d", "e", "f", "g")
x = elements[0]
y = elements[2]
z = elements[-2]

Після чого x матиме значення "a", y - "c", z - "f".

Як перетворити отриманий результат у кортеж?

Жодних спеціальний операцій робити не потрібно. Ти можеш відразу використати return. 

def easy_unpack(elements: tuple) -> tuple:

    x = elements[0]
    y = elements[2]
    z = elements[-2]

    return x, y, z

До речі, кортеж не обов'язково брати в дужки, якщо немає небезпеки сплутати його з іншими змінними. Розв'язок можна записати ще простіше: 

def easy_unpack(elements: tuple) -> tuple:

    return elements[0], elements[2], elements[-2]
