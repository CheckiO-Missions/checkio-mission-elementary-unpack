Вам нужно сделать 2 простых шага: найти 1-й, 3-й и 2-й с конца элементы, а затем вернуть их в виде tuple. Обратите внимание, что в Python нумерация элементов начинается с 0, а не с 1.
Вы могли бы привести наглядный пример?
К примеру, у нас есть tuple ('a', 'b', 'c', 'd', 'e', 'f', 'g') и нам нужно найти его 1-й, 3-й и 2-й с конца элементы. При подсчете элементов с начала tuple - первый элемент будет иметь индекс 0. Если же считать с конца, то "первый" элемент будет иметь индекс "-1", второй - "-2" и так далее.
В коде это будет выглядеть следующим образом: 
elements = ('a', 'b', 'c', 'd', 'e', 'f', 'g')
x = elements[0]
y = elements[2]
z = elements[-2]
После чего x примет значение 'a', y - 'c', z - 'f'.
А как преобразовать полученные результаты в tuple?
Не нужно никаких специальных действий. Вы можете сразу использовать return. 
def easy_unpack(elements):
    x = elements[0]
    y = elements[2]
    z = elements[-2]
    return x, y, z
Или еще проще: 
def easy_unpack(elements):
    return elements[0], elements[2], elements[-2]
