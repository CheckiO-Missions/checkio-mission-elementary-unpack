Ich weiß nicht, wie ich dieses Problem angehen soll.

Sie müssen 2 einfache Schritte machen: Finden Sie das 1., 3. und 2. Element vom Ende her und geben Sie sie dann als Tupel zurück. Beachten Sie, dass in Python die Nummerierung der Elemente mit 0 und nicht mit 1 beginnt.

Könnten Sie ein anschauliches Beispiel geben?

Wir haben zum Beispiel ein Tupel ('a', 'b', 'c', 'd', 'e', 'f', 'g') und müssen das 1., 3. und 2. Element vom Ende her finden. Wenn Sie die Elemente vom Anfang eines Tupels aus zählen, hat das erste Element den Index 0. Wenn Sie vom Ende aus zählen, hat das "erste" Element den Index "-1", das zweite - "-2" und so weiter. Der Code würde wie folgt lauten: 

elements = ('a', 'b', 'c', 'd', 'e', 'f', 'g')
x = elements[0]
y = elements[2]
z = elements[-2]

Danach wird x den Wert 'a', y - 'c', z - 'f' annehmen.

Wie kann man die erhaltenen Ergebnisse in ein Tupel umwandeln?

Es sind keine besonderen Maßnahmen erforderlich. Sie können sofort return verwenden. 

def easy_unpack(elements: tuple) -> tuple:

    x = elements[0]
    y = elements[2]
    z = elements[-2]
    
    return x, y, z

Übrigens ist es nicht notwendig, das Tupel in Klammern einzuschließen, wenn keine Gefahr der Verwechslung mit anderen Variablen besteht. Die Lösung kann noch einfacher geschrieben werden: 

def easy_unpack(elements: tuple) -> tuple:

    return elements[0], elements[2], elements[-2]
