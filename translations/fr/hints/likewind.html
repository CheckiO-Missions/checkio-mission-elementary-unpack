Vous devez effectuer 2 étapes simples: trouver les 1er, 3ème et 2ème éléments à partir de la fin, puis les retourner sous forme de tuple. Faites attention au fait qu'en Python, la numérotation des éléments commence par 0 et non par 1.

Pourriez-vous donner un exemple illustratif?

Par exemple, nous avons un tuple ('a', 'b', 'c', 'd', 'e', 'f', 'g') et nous devons trouver ses 1er, 3e et 2e éléments à partir de la fin. Si l'on compte les éléments à partir du début d'un tuple, le premier élément aura l'indice 0. Si l'on compte à partir de la fin, le "premier" élément aura l'indice "-1", le deuxième - "-2" et ainsi de suite. Le code serait le suivant: 

elements = ('a', 'b', 'c', 'd', 'e', 'f', 'g')
x = elements[0]
y = elements[2]
z = elements[-2]

Ensuite, x prendra la valeur 'a', y - 'c', z - 'f'.

Comment convertir les résultats obtenus en un tuple?

Aucune action particulière n'est requise. Vous pouvez instantanément utiliser return. 

def easy_unpack(elements: tuple) -> tuple:

    x = elements[0]
    y = elements[2]
    z = elements[-2]
    
    return x, y, z

D'ailleurs, il n'est pas nécessaire de mettre le tuple entre parenthèses s'il n'y a pas de risque de confusion avec d'autres variables. La solution peut être écrite encore plus simplement: 

def easy_unpack(elements: tuple) -> tuple:

    return elements[0], elements[2], elements[-2]
